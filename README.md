# Portable handheld gaming device

Uses a Lattepanda Alpha running windows 10 at its core. Paired with the official lattepanda 7' display, a laptop battery I bought on ebay, a speaker driver I managed to get by disassembling a broken bluetooth speaker and an amp lying around I made it 100% portable.

For the controls, it takes advantage of the Lattepandas built-in arduino leonardo chip and connects 2 shoulder buttons, two front buttons, two joysticks and a power button to the GPIO pins. I also implemented a toggleable desktop mode that allows the user to use the joystick and buttons as mouse and mouse button inputs.

Exterior was 3D printed using a Flashforge Finder 3 at home. 3D modeling was done using a mix of Blender and Thinkercad.
